# Frame + Art prompts — all genres (self-contained)

Each genre below has two ready-to-paste prompts. The style block is already written into every prompt — **copy a whole block verbatim** into the image generator; nothing needs to be assembled. Generate the **Hero** (fills the card) and the **Frame** (ornate border, empty center). Save to `public/cards/<slug>/art.png` and `public/cards/<slug>/frame.png`.

Consistency notes: the style block is byte-identical across every genre; the frame always keeps the outer ~12% band decorated and the center empty; each genre has one top-center glyph and one bottom-center glyph for a repeating rhythm.
